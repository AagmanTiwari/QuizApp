Quiz App
A Python-based quiz application where users can register, log in, and take quizzes on various topics. This app saves quiz results for future reference and displays detailed summaries of quiz attempts.

Features
User Registration and Login:

Register with name, email, enrollment number, and password.
Login securely to access the app.
Subjects Covered:

Python Basics
DBMS
Interactive Quiz:

Each subject contains multiple-choice questions with explanations for correct answers.
Immediate feedback on answers.
Result Tracking:

View correct, wrong, and total attempts for each quiz.
Saves results for each user session.
Error Handling:

Handles invalid inputs gracefully.
